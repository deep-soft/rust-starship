# Pure Preset

This preset emulates the look and behavior of [Pure](https://github.com/sindresorhus/pure).

![Screenshot of Pure preset](/presets/img/pure-preset.png)

### Konfiguracja

```sh
starship preset pure-preset -o ~/.config/starship.toml
```
